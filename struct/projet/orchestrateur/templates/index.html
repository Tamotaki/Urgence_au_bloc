Incident actif
ID Incident: sl-inc-8392 · Réseau compromis

Urgence au bloc

Le réseau interne de l'Hôpital Saint-Louis a été compromis par le collectif Solaris. La base de données clinique critique est chiffrée, mettant en péril les plannings chirurgicaux. En tant qu'analyste de crise, vous devez identifier le point d'entrée, analyser les journaux système et récupérer la clé de déchiffrement pour restaurer l'intégrité du service informatique.

Protocole d'investigation

01

Localiser la brèche

Analysez l'application compromise pour découvrir le portail de secours masqué.

02

Forcer l'accès

Exploitez la vulnérabilité d'injection SQL du portail pour récupérer une session valide.

03

Infiltration SSH

Récupérez les identifiants techniques et établissez une connexion shell à distance.

04

Traquer la clé

Fouillez les journaux d'accès Apache pour extraire la clé XOR oubliée par l'attaquant.

05

Restaurer le système

Déchiffrez le fichier de backup à l'aide de la clé et validez la clé système finale.
